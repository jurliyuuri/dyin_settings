# リスターメ語での人名
## 構造
氏族名 - 父称 - 母称 - 名

氏族名，父称，母称はかなり改まった場以外では使われない。

## ありがちな名前
特に記載のない限りは男女どちらの名前にもなる。

|名前|大まかな意味|由来|
|:--:|:--:|:--:|
|altea|"賢い，聡明な"|alto("目") + -ea(形容詞形成接辞，"〜を持つ")||
|elmene|"美しい朝"|el-(接頭辞，"良い，美しい") + mene("朝")||
|elmido|"美しい東方"|el-(接頭辞，"良い，美しい") + mido("朝")||
|junita|"海"|juni("海") + -ta(名詞形成接辞)||
